Напишите функцию, которая принимает два аргумента: массив строк и отдельную строку — и возвращает true, если строка уже содержится в массиве, и false — в противном случае. Например, функция может работать вот так:
arrayContains(["привет", "мир"], "привет");
//=> true
arrayContains(["привет", "мир"], "пока");
//=> false
arrayContains(["привет", "мир", "пока"], "пока");
//=> true
